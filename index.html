How to create a Forum

by Radar

Introduction

I recommend before diving into this tutorial to have a competent enough knowledge of Ruby, even if it is just from a book such as Ruby for Rails: Ruby techniques for Rails developers by David A. Black or Programming Ruby, the latter of which is available freely online.

Conventions

This tutorial uses a few conventions:
  • writing file names in italics
  • putting things that should be typed into a console window in something that resembles this, or I use it also for method names and short one-liner code snippets. Sometimes methods are written by prefixing a word with a hash symbol (#), such as #inspect or #push, sometimes they are not, it depends on the context.
  • putting large blocks of code into areas. So that they can be properly formatted properly. Person class 
       1  class Person
   2    def greet
   3      puts "Hello"
   4    end
   5  end
  • Links look like this
  • When I say /path/to/application I don't mean that exact path! I mean where you just so happened to put your application. For example if you've put it in your home directory in a folder called projects and called it forum, then that path would be ~/projects/forum.
